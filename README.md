# NLP Assignment 2

## Overview
This project demonstrates the implementation of basic Natural Language Processing (NLP) techniques using Python.

## Features
- Text preprocessing
- Tokenization
- Stopword removal
- Lemmatization
- Stemming
- Count Vectorization
- TF-IDF
- Naive Bayes Classification

## Technologies
- Python
- NLTK
- Scikit-learn
- Pandas
- NumPy
